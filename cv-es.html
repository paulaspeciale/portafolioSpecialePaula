Countries App - Proyecto Individual – Henry
Construir una aplicación que consuma información de una API (REST Countries), y a partir de ella guardar cierta información relevante en una base de datos (sequelize - postgres). Cuenta además con funcionalidades que permitan buscar, filtrar y/o ordenar la información según algunos parámetros (por ejemplo, cantidad de población, alfabéticamente).
